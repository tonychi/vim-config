ubuntu-12-vundle
=====================

采用vundle来管理插件，使用于python的开发

支持插件有：
 * NERDTree
 * Ctrlp
 * Powerline
 * taglist
 * snipMate
 * Gundo
 * vim-debug
 * jsbeautify
 * Zencoding

使用习惯的特殊设置有：
 * 显示行号，宽度为5
 * 显示搜索关键字的高亮
 * 显示状态栏
 * 鼠标可用
 * 去除工具栏显示
 * 高亮当前行，背景颜色#000000
 * 配色方案采用desert
 * 共享系统剪贴板
 * python 使用缩进方式控制折叠，折叠级别99, 使用空格键打开和关闭
   折叠, 自动切换
 * 开启语法侦测
 * 增加窗口间切换的快捷键
